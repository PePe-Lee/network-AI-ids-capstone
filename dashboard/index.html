Risk Level 설명

🟢 BENIGN-like
정상 baseline과 매우 유사한 flow.
모델 score가 정상 분포 안에 있고
anomaly detector가 반응하지 않은 상태.
🟡 BORDERLINE
정상으로 판단되지만 score가
정상 분포의 상위 구간에 가까운 flow.
정상일 가능성이 높지만 약간 특이한 flow.
🟠 SUSPICIOUS
AE, LOF, Fusion 중 하나 이상이
threshold를 넘어 anomaly로 판단한 flow.
공격일 수도 있고 정상 outlier일 수도 있음.
🔴 ATTACK-like
여러 모델이 동시에 anomaly로 판단했거나
behavior rule이 탐지한 강한 이상 flow.
공격 가능성이 높은 flow.
